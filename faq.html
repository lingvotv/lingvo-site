Frequently Asked Questions

  • Are you using one of the supported services to play the movie?
  • Are you playing the movie in Chrome Browser on your desktop computer or laptop?
  • Are the subtitles turned on? Do you see them changing in the movie?
  • Have you installed the Chrome Extension in the same browser where you opened the movie?
  • Have you opened the movie after you installed the extension? If not, please reload the window with the movie.
  • Have you entered email in the Chrome Extension?
  • Have you used the same email to signup/login into LingvoTV Web App on your phone?
  • Have you checked the internet connection on both devices?
LingvoTV Chrome Extension is a small program you can only install on a Chrome Browser on a desktop computer. It will grab subtitles appearing on the screen as you watch the movie and send them to LingvoTV Web App.

It is required to install it because otherwise, LingvoTV Web App won't receive subtitles.

Open LingvoTV site on a desktop Chrome Browser, click on the "Chrome Extension" button, and it will take you to the official Chrome Web Store, then click on "Add to Chrome" to install.
LingvoTV Web App is a site located under app.lingvo.tv, where you translate the words while watching the movie. We built it for mobile devices, but you can open it in any browser on any device like a tablet or even on the same desktop computer where you watch the movie. It is essential, though, that it is convenient for you to see both the movie and the Web App at the same time.

The idea is that you can watch the movie and translate unknown words quickly without interruption.
LingvoTV Web App doesn't need installation. It is instantly available on any device if you go to app.lingvo.tv in any browser or click on "Web App" on the site.
Chrome Extensions reads the subtitles from the movie and sends them to the Web App, where you can see and translate them.
Right now, we built only a Chrome Extension, but in the future, we can offer an extension for other browsers and devices as well.

It is essential to understand that LingvoTV Web App works in any browser on any device. Only LingvoTV Extension requires Desktop Chrome, so that's where you have to watch the movie at the moment.
LingvoTV uses Google Translate as a translation service, so LingvoTV will translate the same languages Google Translate does.
LingvoTV does not provide subtitles, as it only sends them from the movie to the Web App.

Subtitles languages provided by services like Netflix depend on your location and the movie itself.
If you wish LingvoTV had a feature, it currently doesn't - don't hesitate to contact us and describe your use case.
